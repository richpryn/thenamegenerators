# Generator Article Template

When creating a new generator article, always include these two sections before the FAQs:

## Required Sections

### 1. Well-Known [Name Type] Names in Literature and Media

This section should include:
- 5-10 iconic examples from literature, games, films, or other media
- Brief explanation of what makes each name effective
- Discussion of naming conventions demonstrated by these examples
- Guidance on how to apply these principles to your own names

**Example heading format:**
- "Well-Known Elf Names in Literature and Media"
- "Well-Known Dwarf Names in Literature and Media"
- "Well-Known Vampire Names in Literature and Media"

### 2. The Meaning and Derivation of [Name Type] Names

This section should include:
- Etymology and linguistic roots
- Symbolic meanings and cultural references
- Common naming patterns and conventions
- How name elements contribute to character identity
- Examples showing how meanings combine to create effective names

**Example heading format:**
- "The Meaning and Derivation of Elf Names"
- "The Meaning and Derivation of Dwarf Names"
- "The Meaning and Derivation of Vampire Names"

## Section Placement

These sections should be placed **after** the main content sections and **before** the FAQs:

```json
{
  "article": {
    "sections": [
      {
        "heading": "Main Content Section 1",
        "content": "..."
      },
      {
        "heading": "Main Content Section 2",
        "content": "..."
      },
      {
        "heading": "Well-Known [Name Type] Names in Literature and Media",
        "content": "..."
      },
      {
        "heading": "The Meaning and Derivation of [Name Type] Names",
        "content": "..."
      }
    ],
    "faqs": [...]
  }
}
```

## Content Guidelines

### Well-Known Section Content:
- Focus on memorable, recognizable examples
- Explain why each name is effective
- Show variety in naming conventions
- Connect examples to naming principles

### Meaning and Derivation Section Content:
- Cover etymology and linguistic roots
- Explain symbolic meanings
- Discuss cultural and historical references
- Show how elements combine to create meaning
- Provide examples of name analysis

## Notes

- Both sections are required for all generators
- Content should be specific to the name type
- Use proper HTML formatting for lists and paragraphs
- Ensure content is accurate and well-researched
- Keep content engaging and informative
