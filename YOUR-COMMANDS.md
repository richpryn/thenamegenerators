# Your GitHub Commands

## Your GitHub Username: `richpryn`

## Step 1: Create Repository on GitHub

1. Go to: **https://github.com/new**
2. Repository name: `name-generators`
3. Description: "Free online name generators for D&D, writing, gaming, and creative projects"
4. Visibility: **Public** ✅
5. **DO NOT** check "Initialize with README"
6. Click **"Create repository"**

## Step 2: Run These Commands

**After creating the repository, run these exact commands:**

```bash
git remote add origin https://github.com/richpryn/name-generators.git
git push -u origin main
```

**Or run the script:**
```bash
./PUSH-COMMANDS.sh
```

## Step 3: Enable GitHub Pages

1. Go to: https://github.com/richpryn/name-generators
2. Click **Settings** (top menu)
3. Click **Pages** (left sidebar)
4. Source: **Deploy from a branch**
5. Branch: **main**
6. Folder: **"/ (root)"**
7. Click **Save**

Your site will be live at: `https://richpryn.github.io/name-generators/`

## What Happens Next

After pushing:
- ✅ Your code will be on GitHub
- ✅ GitHub Pages will build your site
- ✅ Site will be accessible at the GitHub Pages URL
- ⏭️ Then add custom domain (Step 2 in deployment guide)
